# Approach
1. Identify longest special substrings. For example: `aaabbbdaaa` --> `aaa` `bbb` `d` `aaa`
2. Store their length in `ArrayList[26] set` corresponding to character. For example: `aaa` `bbb` `d` `aaa` --> `set[0] = [3, 3]`, `set[1] = [3]`, `set[3] = [1]`
3. For every character indentify the largest three length of substring. `t1 t2 t3`, where`t1 <= t2 <= t3`.
4. * If `t1 != 0 and t2 != 0 and t3 != 0` then the length of longest trice special substring is `max(t1, max(min(t2, t3 - 1), t3 - 2))`.
* If `t1 == 0 and t2 != 0 and t3 != 0`then the length of longest trice special substring is `t3 - 1`.
* If `t1 == 0 and t2 == 0 and t3 != 0`then the length of longest trice special substring is `t3 - 2`.

# Complexity
- Time complexity:
  $$O(N)$$

- Space complexity:
  $$O(N)$$

# Code
```java []
import java.util.*;
class Solution {
    public int maximumLength(String s) {
        ArrayList[] set = new ArrayList[26];
        for (int i = 0; i < 26; i++) set[i] = new ArrayList<>();

        for (int i = 0; i < s.length();) {
            int b = i + 1;
            while (b < s.length() && s.charAt(b) == s.charAt(i)) b++;
            set[(int) (s.charAt(i) - 'a')].add(b - i);
            i = b;
        }

        int max = -1;
        for (int i = 0; i < 26; i++) {
            int t1 = 0, t2 = 0, t3 = 0;
            for (int j = 0; j < set[i].size(); j++) {
                if (((Integer) set[i].get(j)) > t1) t1 = ((Integer) set[i].get(j));
                if (t1 > t2) {
                    int temp = t1;
                    t1 = t2;
                    t2 = temp;
                }
                if (t2 > t3) {
                    int temp = t3;
                    t3 = t2;
                    t2 = temp;
                }
                if (t1 > t2) {
                    int temp = t1;
                    t1 = t2;
                    t2 = temp;
                }

                if (t1 > 0 || (t2 > 0 && t3 > 1) || (t3 > 2)) {
                    
                    if (t2 == 0) max = Math.max(max, t3 - 2);
                    else if (t3 == 0) max = Math.max(max, t3 - 1);
                    else max = Math.max(max, Math.max(Math.max(t1, Math.min(t3 - 1, t2)), t3 - 2));
                }
            }

        }
        return max;
    }
}
```
