# Approach

In this solution, you are not going to learn optimization but method utilization in java.

### Updating by idxHolder

Since the attributes of each item are distributed in different arrays, it may be difficult to change all arrays at the same time.
$$\rightarrow$$ We can operate on `idxHolder`, which is initialized to be `{ 0, 1, 2, 3, 4... }`. After sorting and removing, we can still keep track of all the attributes by `code[idxHolder[i]]`, `businessLine[idxHolder[i]]`, `isActive[idxHolder[i]]`.

### Stream API

For this array problem, we can leverage the Stream API with `filter` and `sorted` using lambdas to perform all operations.

# Complexity
- Time complexity:
  $$O(n\log n)$$

- Space complexity:
  $$O(n)$$

# Code
```java []
class Solution {
    public List<String> validateCoupons(String[] code, String[] businessLine, boolean[] isActive) {
        // initialize idxHolder
        Integer[] idxHolder = new Integer[businessLine.length];
        Arrays.setAll(idxHolder, i -> i);

        // valid businessLine
        Set<String> validBus =
            Set.of("electronics", "grocery", "pharmacy", "restaurant");

        // stream
        return Arrays.stream(idxHolder)
            .filter(i ->
                isActive[i] &&
                validBus.contains(businessLine[i]) &&
                code[i].matches("^[a-zA-Z0-9_]+$")
            )
            .sorted((i1, i2) -> 
                !businessLine[i1].equals(businessLine[i2]) ?
                businessLine[i1].compareTo(businessLine[i2]) :
                code[i1].compareTo(code[i2])
            )
            .map(i -> code[i])
            .collect(Collectors.toList());
    }
}
```
