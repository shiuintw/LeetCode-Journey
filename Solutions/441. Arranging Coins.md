# Approach
* Find ans = x such that: `(x * x + x) / 2 <= n`.
* Because the interval `[1, ans]` is increasing, so we can use binary search with respect to `x`.
* Keep in mind that `x * x` may exceed the `Integer.MAX_VALUE`, so use `long` instead.

# Complexity
- Time complexity:
  $$O(Log(N))$$

- Space complexity:
  $$O(1)$$

# Code
```java []
class Solution {
    public int arrangeCoins(int n) {
        long layer = 0;
        for (long jump = n; jump > 0; jump /= 2) {
            while (((layer + jump) * (layer + jump) + (layer + jump)) / 2 <= n) {
                layer += jump;
            }
        }
        return (int) layer;
    }
}
```
