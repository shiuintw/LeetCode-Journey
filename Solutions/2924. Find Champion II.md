# Approach
1. Compute the in-degree of every node by iterating over the edges list.
2. Identify nodes with an in-degree of 0. If more than one such node exists, return -1. If exactly one such node exists, it is the champion.

# Complexity
- Time complexity:
  $$O(N)$$

- Space complexity:
  $$O(N)$$

# Code
```java []
class Solution {
    public int findChampion(int n, int[][] edges) {
        int[] indeg = new int[n];
        for (int[] e : edges) indeg[e[1]]++;
        
        int champion = -1;
        for (int i = 0; i < n; i++) {
            if (champion == -1 && indeg[i] == 0) {
                champion = i;
            } else if (champion != -1 && indeg[i] == 0) return -1;
        }
        return champion;
    }
}
```
