# Approach
* Since a long has 64 bits for storage and the maximum value is 500, we need a `long[8]` array (64 × 8 = 512 bits) in total.
* Each number is stored at `bitset[num / 64][num % 64]`.
* Our goal is to check whether the frequency of all numbers is even, so we use XOR.

# Complexity
- Time complexity:
  $$O(N)$$

- Space complexity:
  $$O(\text{MaxValue} / 64)$$ --> $$O(1)$$

# Code
```java []
class Solution {
    public boolean divideArray(int[] nums) {
        long[] bitset = new long[8];
        for (int num : nums) {
            bitset[num / 64] ^= (1L << num % 64);
        }
        for (long b : bitset) {
            if (b != 0) return false;
        }
        return true;
    }
}
```
