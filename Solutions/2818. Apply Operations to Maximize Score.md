# Approach
1. find all primes by using Sieve of Eratosthenes
```java []
// find all primes (about 66 primes when n <= 10 ^ 2.5)
boolean[] isPrime = new boolean[(int) Math.sqrt(maxVal) + 1];
Arrays.fill(isPrime, true);
for (int i = 2; i * i < isPrime.length; i++) {
    if (isPrime[i]) {
        for (int j = i * i; j < isPrime.length; j += i) {
            isPrime[j] = false;
        }
    }
}
ArrayList<Integer> primes = new ArrayList<>();
for (int i = 2; i < isPrime.length; i++) {
    if (isPrime[i]) primes.add(i);
}
```

2. calculate prime score
```java []
// calculate the prime score
int[] primeScore = new int[n];
for (int i = 0; i < n; i++) {
    int num = nums.get(i);
    for (int prime : primes) {
        if (prime * prime > num) break;
        if (num % prime == 0) {
            primeScore[i]++;
            while (num % prime == 0) num /= prime;
        }
    }
    if (num > 1) primeScore[i]++;
}
```

3. find left and right s.t. `primeScore[left] <= primeScore[i] >= primeScore[right]` by using monotonic stack
```java []
// find left <= i <= right s.t.
// primeScore[left] <= primeScore[i] >= primeScore[right];
int[] lefts = new int[n];
int[] rights = new int[n];
ArrayDeque<Integer> mono = new ArrayDeque<>();
// find right
for (int i = 0; i < n; i++) {
    while (!mono.isEmpty() &&
    primeScore[mono.peekLast()] < primeScore[i]) {
        lefts[mono.pollLast()] = i;
    }
    mono.offerLast(i);
} 
while (!mono.isEmpty()) lefts[mono.pollLast()] = n;

// find left
for (int i = n - 1; i >= 0; i--) {
    while (!mono.isEmpty() &&
    primeScore[mono.peekLast()] <= primeScore[i]) {
        rights[mono.pollLast()] = i;
    }
    mono.offerLast(i);
}
while (!mono.isEmpty()) rights[mono.pollLast()] = -1;
```

4. find range (number of subarray dominant by index i)
```java []
// find range = (i - left) * (right - i)
long[] ranges = new long[n];
for (int i = 0; i < n; i++) {
    ranges[i] =  ((long) (i - lefts[i]) * (rights[i] - i));
}
```

5. use an `indices` array to store the indexes of `nums`, then sort indices based on the values in nums
```java []
// sort by nums
Integer[] indices = new Integer[n];
Arrays.setAll(indices, i -> i);
Arrays.sort(indices, (i1, i2) -> nums.get(i2) - nums.get(i1));
```

6. calculate the answer
```java []
// calculate ans : res *= num ^ range % mod
long res = 1;
for (int idx : indices) {
    if (k <= 0) break;
    res = (res * expoCal(nums.get(idx), Math.min(k, ranges[idx]), mod)) % mod;
    k -= ranges[idx];
}
return (int) res;
```

7. fast way to calculate `A ^ B % C`
```java []
private int expoCal(long A, long B, int C) { // A ^ B % C
    long res = 1;
    long power = A % C;
    while (B > 0) {
        if (B % 2 == 1) {
            res = (res * power) % C;
        }
        power = (power * power) % C;
        B /= 2;
    }
    return (int) res;
}
```

# Complexity
- Time complexity:
  $$O(Nlog(N)log(\text{MAX\_VALUE}))$$

- Space complexity:
  $$O(N)$$

```java []
class Solution {
    private int expoCal(long A, long B, int C) { // A ^ B % C
        long res = 1;
        long power = A % C;
        while (B > 0) {
            if (B % 2 == 1) {
                res = (res * power) % C;
            }
            power = (power * power) % C;
            B /= 2;
        }
        return (int) res;
    }

    public int maximumScore(List<Integer> nums, int k) {
        final int mod = 1000000007;
        final int n = nums.size();

        // find max value
        int maxVal = 0;
        for (int i : nums) maxVal = Math.max(i, maxVal);

        // find all primes (about 66 primes when n <= 10 ^ 2.5)
        boolean[] isPrime = new boolean[(int) Math.sqrt(maxVal) + 1];
        Arrays.fill(isPrime, true);
        for (int i = 2; i * i < isPrime.length; i++) {
            if (isPrime[i]) {
                for (int j = i * i; j < isPrime.length; j += i) {
                    isPrime[j] = false;
                }
            }
        }
        ArrayList<Integer> primes = new ArrayList<>();
        for (int i = 2; i < isPrime.length; i++) {
            if (isPrime[i]) primes.add(i);
        }

        // calculate the prime score
        int[] primeScore = new int[n];
        for (int i = 0; i < n; i++) {
            int num = nums.get(i);
            for (int prime : primes) {
                if (prime * prime > num) break;
                if (num % prime == 0) {
                    primeScore[i]++;
                    while (num % prime == 0) num /= prime;
                }
            }
            if (num > 1) primeScore[i]++;
        }

        // find left <= i <= right s.t.
        // primeScore[left] <= primeScore[i] >= primeScore[right];
        int[] lefts = new int[n];
        int[] rights = new int[n];
        ArrayDeque<Integer> mono = new ArrayDeque<>();
        // find right
        for (int i = 0; i < n; i++) {
            while (!mono.isEmpty() &&
            primeScore[mono.peekLast()] < primeScore[i]) {
                lefts[mono.pollLast()] = i;
            }
            mono.offerLast(i);
        } 
        while (!mono.isEmpty()) lefts[mono.pollLast()] = n;

        // find left
        for (int i = n - 1; i >= 0; i--) {
            while (!mono.isEmpty() &&
            primeScore[mono.peekLast()] <= primeScore[i]) {
                rights[mono.pollLast()] = i;
            }
            mono.offerLast(i);
        }
        while (!mono.isEmpty()) rights[mono.pollLast()] = -1;

        // find range = (i - left) * (right - i)
        long[] ranges = new long[n];
        for (int i = 0; i < n; i++) {
            ranges[i] =  ((long) (i - lefts[i]) * (rights[i] - i));
        }

        // sort by nums
        Integer[] indices = new Integer[n];
        Arrays.setAll(indices, i -> i);
        Arrays.sort(indices, (i1, i2) -> nums.get(i2) - nums.get(i1));

        // calculate ans : res *= num ^ range % mod
        long res = 1;
        for (int idx : indices) {
            if (k <= 0) break;
            res = (res * expoCal(nums.get(idx), Math.min(k, ranges[idx]), mod)) % mod;
            k -= ranges[idx];
        }
        return (int) res;
    }
}
```
