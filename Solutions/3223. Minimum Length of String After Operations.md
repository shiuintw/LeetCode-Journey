# Approach
1. To count the total frequency of each character, you might consider using `Map<Character, Integer>` or `int[]`.
2. However, we only care about whether the frequency is `0 or 1 or 2`.
3. So we can think of using two int variables : `iset1, iset2` to store the value in the `character - 'a'`-th bit by taking bit manipulation : `iset1 |= (1 << (character - 'a'))`.

# Complexity
- Time complexity:
  $$O(N)$$


- Space complexity:
  $$O(1)$$
  Since iset1, iset2 both take 32 bits space which is fixed and constant complexity.

# Code
``` c []
int minimumLength(char* s) {
    int iset1 = 0, iset2 = 0;
    for (int i = 0; s[i] != NULL; i++) {
        int pos = 1 << (s[i] - 'a');
        if ((iset1 & pos) != 0) {
            if ((iset2 & pos) != 0) {
                iset2 ^= pos;
            } else iset2 |= pos;
        } else iset1 |= pos;
    }

    int total = 0;
    for (int i = 0; i < 26; i++) {
        int pos = 1 << i;
        if ((iset2 & pos) != 0) total += 2;
        else if ((iset1 & pos) != 0) total++;
    }
    return total;
}
```

```java []
class Solution {
    public int minimumLength(String s) {
        int iset1 = 0, iset2 = 0;
        for (int i = 0; i < s.length(); i++) {
            int pos = 1 << (s.charAt(i) - 'a');
            if ((iset1 & pos) != 0) {
                if ((iset2 & pos) != 0) {
                    iset2 ^= pos;
                } else iset2 |= pos;
            } else iset1 |= pos;
        }

        int total = 0;
        for (int i = 0; i < 26; i++) {
            int pos = 1 << i;
            if ((iset2 & pos) != 0) total += 2;
            else if ((iset1 & pos) != 0) total++;
        }
        return total;
    }
}
```
